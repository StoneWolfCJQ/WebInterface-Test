# 简介  
A simple UI for IO check on local and remote devices
可以在本地和远程设备上校验IO的软件  
`注1：目前只支持ACS和三菱系列的PLC，并且需要安装SPiiPlus MMI Application Studio 2.50相应软件及三菱的MX COMPONENT（我用的是4.16）`  
`注2：这只是一个试验版程序，代码写得比较乱，没有注释，性能也很差，以后有时间会重写`
